Stylelint rules for my projects. Feel free to use these conventions.

## Usage

Just add `@miszo/stylelint-config` to yours `extends` section in stylelint config.

```javascript
module.exports = {
  extends: [
    '@miszo/stylelint-config '
  ]
};
```
